# Living Mirror Starter Interview

Use when the user has little or no prepared data.

## Goal

What do you want to understand first?

- [ ] Myself generally
- [ ] A relationship pattern
- [ ] A work/creator pattern
- [ ] A decision
- [ ] A recent change

## Questions

1. What is the pattern you keep meeting again?
2. When did it become obvious?
3. What do you usually call this pattern in your own words?
4. What are three examples?
5. What are two counter-examples?
6. Which relationship or context makes it stronger?
7. Which context makes it disappear?
8. What would you hate for the mirror to assume about you?
9. What would be useful to try in the next seven days?
10. What should remain private?
