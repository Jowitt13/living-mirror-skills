# 特殊时期标注规则

## 规则

每个时间段/主题的蒸馏开头，强制标注是否包含特殊时期。

特殊时期包括：

- 考试周、期末周、备考冲刺期
- 暑假、寒假、长假
- 生病期、失眠期、情绪危机期
- 重大事件期，如搬家、分手、升学、工作变动、家庭事件
- 热恋初期、冲突高峰期等关系阶段性异常

## 判断纪律

1. 凡涉及"下降/减少/边缘化/变弱/失去兴趣"的判断，必须先排除时间因素。
2. 特殊时期数据不计入正式稳态画像，或明确标注"特殊期数据"。
3. 可以分析特殊时期的反应模式，但不要把它写成长期人格结论。
4. 同一维度至少需要特殊期之外的数据交叉验证。

## 报告写法

```markdown
## 特殊时期标注
- 本时间段是否包含特殊时期：是/否
- 类型：考试周/假期/生病期/重大事件/关系高浓度期
- 受影响维度：目标与方向、情绪、作息等
- 处理方式：不计入正式画像 / 仅作为特殊期反应 / 需后续验证
```

## 典型错误

- 用期末周疲惫判断长期热情弱。
- 用暑假学业词频低判断学业被边缘化。
- 用热恋期凌晨高频判断稳定依恋模式。
- 用备考期崩溃表达判断长期情绪基线。
