# 关键词使用规则

## 关键词能做什么

关键词只用于发现值得阅读的对话段、构建候选样本、生成描述性统计。

可以用于：

- 找出某主题相关消息。
- 发现高密度时间段。
- 辅助选择长消息和上下文窗口。
- 做 artifact 级描述。

## 关键词不能做什么

不能直接用于定性：

- 关系性质
- 情绪状态
- 依恋类型
- 人格标签
- 目标优先级
- 价值观强弱

## 多义词风险

- "哭"可能是"哭死/笑哭了/哭笑不得"。
- "怕"可能是"怕不怕/你怕啥/不怕"。
- "分"可能是"分享/部分/分钟/十分"。
- "爱"可能是称呼、玩笑、歌词、引用。
- "钱/买/花"可能是讨论、吐槽、计划，不等于实际消费行为。

## 使用流程

1. 用关键词筛选候选消息。
2. 保留前后上下文。
3. 读原文确认语义。
4. 验证 sender。
5. 只把关键词统计写成 artifact，不作为唯一结论。

## 报告写法

```markdown
- 证据：[artifact] 关键词命中：哭 58 次 / 怕 32 次。
- 限制：关键词存在语义噪音，只用于定位对话段，不用于情绪定性。
- 定性依据：以下已读原文片段...
```
